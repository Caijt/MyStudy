### 初始化项目

yarn init

### 添加依赖包

yarn add <package>

yarn add <package@version>

yanr add <package> --dev

yarn global add <package>

### 更新依赖包vue 

yarn update <package>

### 移除依赖包

yarn remove <package>

yarn  global remove <package>

### 安装项目全部依赖

yarn



### 解决无法安装node-sass

```bash
yarn config set sass-binary-site http://npm.taobao.org/mirrors/node-sass
```
